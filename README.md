## Базовый уровень сложности
> ### 1. Ввод данных
> - Пользователь должен ввести время для будильника в формате
    "ЧЧ:ММ". Также у него есть возможность выбрать звуковое
    уведомление или текстовое сообщение

> ### 2. Цикл работы программы
>  - Программа должна постоянно проверять текущее время
    и сравнивать его с установленным временем будильника.
> - Если время совпало, будильник срабатывает

> ### 3. Вывод данных
>  - При срабатывании будильника программа должна вывести
    сообщение "Время вставать!" или проиграть звук
> ### 4. Дополнительные возможности
> - Возможность установить несколько будильников (для
    этого использовать список/словарь для хранения времени
    будильников).
> - Будильник должен работать до тех пор, пока
    не будет остановлен пользователем.
